## O projektu

_Krátce popište svůj projekt – čemu se věnuje, co nabízí a jaký problém řeší?_

`Jedná se o informativní web o mém úžasném fotbalovém týmu v nejnižší české ffotbalové lize.
Web bude mít základní info, soupisku, vedení, novinky, kalendář zápasů/akcí a kontakty`

## Konkurence / Inspirace

_Uveďte konkurenční projekty. Co se vám na nich líbí, co ne? V čem se můžete inspirovat?_
